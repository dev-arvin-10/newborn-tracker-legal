# Newborn Tracker Legal Documents

This repository contains the legal documents for the Newborn Tracker iOS app, including:

- [Privacy Policy](privacy-policy.html)
- [Terms of Service](terms-of-service.html)
- [Support & FAQ](support.html)

## Hosting Instructions

These documents are designed to be hosted on GitHub Pages. To set up hosting:

1. Create a new GitHub repository (e.g., `newborn-tracker-legal`)
2. Upload all files from this `docs/` folder to the repository
3. Enable GitHub Pages in repository settings
4. Set source to "Deploy from a branch" and select "main" branch
5. Your documents will be available at: `https://yourusername.github.io/repository-name/`

## Alternative Hosting Options

You can also host these documents on:
- Netlify (drag and drop the docs folder)
- Vercel
- Your own web server
- Any static site hosting service

## Updating URLs

After hosting, update the URLs in `NewbornTracker/src/utils/subscriptionConstants.ts` with your actual domain.
